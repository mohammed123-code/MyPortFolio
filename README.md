# Santhu Mohammed M - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, education, and achievements as a Computer Science Engineering student specializing in AI/ML.

## 🌟 Features

### Design & User Experience
- **Modern & Responsive Design**: Clean, professional layout that works perfectly on all devices
- **Smooth Animations**: Engaging animations and transitions throughout the site
- **Interactive Elements**: Hover effects, skill bars, and dynamic content
- **Mobile-First**: Fully responsive design with mobile navigation
- **Accessibility**: Proper semantic HTML and keyboard navigation support

### Sections Included
1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Me**: Personal introduction and contact information
3. **Education**: Academic background with institutions and achievements
4. **Projects**: Detailed showcase of technical projects with tech stacks
5. **Skills**: Visual skill bars for programming languages and tools
6. **Certifications**: Professional certifications and training programs
7. **Achievements**: Awards and recognitions
8. **Coding Profiles**: LeetCode, HackerRank, and Skill Rack statistics
9. **Contact**: Multiple ways to get in touch

### Technical Features
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Active Navigation**: Highlights current section in navigation
- **Back to Top Button**: Easy navigation to the top of the page
- **Loading Animations**: Skill bars animate when scrolled into view
- **Parallax Effects**: Subtle parallax scrolling in hero section
- **Copy to Clipboard**: Click to copy contact information
- **Intersection Observer**: Efficient scroll-based animations

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. Download or clone the repository
2. Open `index.html` in your web browser
3. The portfolio is ready to use!

### File Structure
```
Portfolio/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and responsive design
├── script.js           # JavaScript functionality and animations
└── README.md           # This file
```

## 🎨 Customization

### Personal Information
Update the following sections in `index.html`:
- Personal details in the hero section
- Contact information in the about and contact sections
- Education details
- Project descriptions and tech stacks
- Skills and proficiency levels
- Certifications and achievements

### Styling
Modify `styles.css` to customize:
- Color scheme (primary colors, gradients)
- Typography (fonts, sizes, weights)
- Layout spacing and grid systems
- Animation timings and effects
- Responsive breakpoints

### Functionality
Enhance `script.js` to add:
- Additional animations
- Form handling for contact
- Portfolio filtering
- Dark mode toggle
- Custom interactions

## 📱 Responsive Design

The portfolio is fully responsive with breakpoints for:
- **Desktop**: 1200px and above
- **Tablet**: 768px to 1199px
- **Mobile**: Below 768px
- **Small Mobile**: Below 480px
